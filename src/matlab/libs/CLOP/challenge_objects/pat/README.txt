This directory contains pattern recognition algorithms for binary classification problems.
gentleboost: GentleBoost
kridge: kernel ridge regression
naive: naive Bayes
neural: on hidden layer neural net
rf: random forest
svc: support vector classifier

Notes: 
- to run RF, you must first install R, see http://www.r-project.org/
- to run SVC under Linux, you must compile the C code, see
../packages/libsvm-mat-2.8-1
